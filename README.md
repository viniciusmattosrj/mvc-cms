## Requerimentos

- Install <a href="https://docs.docker.com/install/">Docker</a>
- Install <a href="https://docs.docker.com/compose/install/">docker-compose</a>
- Mysql >= 5.7 ou Postgres >= 11
- Composer
- PHP &ge; 7.3.22

### Tecnologias Utilizadas

- PHPUnit &ge; 7.5.\*
